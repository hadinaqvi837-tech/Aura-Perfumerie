# Aura Perfumerie — Run locally

This repository contains a small Vite + React demo for the Aura Perfumerie storefront.

## Run Locally

**Prerequisites:** Node.js (LTS)

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` (if you plan to call the Gemini API)
3. Run the app:
   `npm run dev`
